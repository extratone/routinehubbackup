Description

This shortcut uses the Snap.as API to upload single images to Snap.as. At install, you’ll be required to provide your Write.as/Snap.as API token, which you can acquire with my Siri Shortcut or manually with the Write.as API.

Input: Image files/photos from Apple Photos Output: a Snap.as embed URL.
