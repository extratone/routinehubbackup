Description

This shortcut assumes you’ve used Safari 15’s “copy links” feature to copy the rich text links of your open tabs to the clipboard. It extracts the raw URLs from said links via Make Markdown from Rich Text followed by a Get URLs from Input action and stores said URLs in the Data Jar list specified at install.

The companion shortcut - Restore Safari Tabs - can be used to reopen the links you've stored in Safari.
