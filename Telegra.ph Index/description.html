Description

This shortcut uses a single getPageList request using the Telegra.ph API to retrieve a list of titled hyperlinks to one's Telegra.ph posts.

At installation, you'll be asked to specify your Telegra.ph API token, which I was personally able to find by logging in to my account in a desktop browser so I could find it in Developer Tools. This Stack Overflow answer comprehensively describes the process.
