Description

Please note: as of the original release date, iOS16 Developer Beta is required for this Shortcut to run.*

An extremely rudimentary iOS16-only shortcut that utilizes the update's new Remove Background from Image action to automatically convert image data behind the (perceived) subject to transparency . It then ensures said image will retain said transparency with a Convert Image action before saving the (copied) background-less output to an iCloud Photos folder specified at install.
