Description

Note: This shortcut requires a Mastodon application with write permissions. You can create one at [yourinstance]/settings/applications. See the documentation for more information.

This shortcut creates a single text-only Mastodon post on an instance of your specification via the Mastodon API. Upon installation, you will be required to specify your instance and provide a client access token.
