Description

Looking for a way to keep track off all your RoutineHub downloads?
This is the shortcut you need, it has the following features.

  • Automatic saving off each report ( this can be turned off )
  • Clear indicators of new downloads and new hearts
  • Indicators for removed hearts
  • Supports landscape and dark mode

Saving and removing reports

Reports can be saved on each run, if you want to delete them that is also possible. There are options included to delete them all at the same time or choose which one to delete.
