Description

This shortcut will convert EPUB books and can optionally send them to your Kindle via its email address.

Configuration

CloudConvert

This shortcut makes use of the the CloudConvert service and requires you to sign up for a free, personalized API key.

To get your API key:

  1. Register an account at CloudConvert.
  2. Visit the API Dashboard and create your API key.

Send to Kindle by email

Kindle offers a free service to send personal documents to your Kindle by email.

To enable this optional service and setup your Kindle email address, see the following instructions:

Send to Kindle by E-mail

Finding free EPUB books

The Project Gutenberg hosts a large number of classic books and other works for which U.S. copyright has expired.
