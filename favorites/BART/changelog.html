2.5 2 years, 4 months ago
iOS 13
- In the Help, added notes regarding testing with iOS Developer beta 4.
- Removed the 'Show Shortcuts Preview before Restore' option. This was added in version 2.2 but is no longer needed because the root cause of the issue that motivated the option has been identified: With iOS Developer beta 4 (and likely earlier iOS 14 beta versions), Text variables in 'If' actions can sometimes lose their 'Type' as 'Text' and revert to 'Contact'. This can occur when the 'If' action follows a 'Choose from List' action and that action is using list items that are formatted using vCard blocks
- The above issue was addressed by adding a 'Text' action between the 'Choose from List' and 'If' actions.
