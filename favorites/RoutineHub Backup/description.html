Description

This shortcut mostly exists because of the frequent downtime of RoutineHub (this week has been particularly painful). So... thanks for the inspiration? üòã

This shortcut allows you to easily save the description and version history pages of all the shortcuts created by a user on RoutineHub. The saved pages include direct iCloud links and are a simplified version of the pages on the RoutineHub website.

The shortcut asks a few questions and then spends a few minutes downloading the pages and generating the files. The files are saved in iCloud Drive under the "Shortcuts/RoutineHub Backup" folder, and are organized by user and shortcut name. The shortcut also has the option to save the user's favorite shortcuts in a separate folder.

The shortcut checks at the beginning to make sure that RoutineHub is available and that the user's account exists. Also, since version 1.0.2, the shortcut skips any shortcut with an existing backup.

This shortcut is useful both for backing up your own RoutineHub account, and for backing up any other RoutineHub account.
