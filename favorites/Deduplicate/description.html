Description

This shortcut silently deduplicates a list of items received as input. The output is a list of unique items. These items can be texts, photos, videos or files.

  • If it’s run without input, it will work as a demo: the user will be asked to select a data type and a few items of that type, and the result of the deduplication will be shown
  • The shortcut compares items using MD5 hashes, so two items will be considered equals only if their contents as files are identical bit by bit

Note: For lines of text the shortcut De-Duplicate List by @EggsistentialBread is faster.
