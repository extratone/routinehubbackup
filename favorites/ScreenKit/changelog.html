1.2 2 years, 3 months ago
iOS 13
This version introduces some stability improvements and two new big features.

New features:
- Added Install Update screen, which lets the developer show the user a wonderful and fresh update prompt, with release notes and download button. All this stuff is generated automatically thanks to the great power of the JavaScript. You only need to enter the RoutineHub ID of your shortcut and ScreenKit will do all the work once again.
- Added What’s New screen, which is meant for showing once after each update and inform the user about the new stuff introduced with that update. Also here, ScreenKit does all the work using JavaScript while you only have to enter the RoutineHub ID of your Shortcut.
