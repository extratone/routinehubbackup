Description

Total Size Of Folder is a shortcut to know the total size of a folder. It is also the fastest, using a-Shell and the du command.

How it works

  • Open the shortcut and select the folder you want to measure.
  • The shortcut will use a-Shell and the 'du' command to get the apparent total size of a folder (and also of its files and subfolders, including invisible elements).
  • Then it will show you a pretty HTML table with the total size followed by the other elements, sorted by size. You can copy, export or print it.

Companions

  • If you need to compute the size of a group of files, you can use my other shortcut Total Size Of Files.

REQUIREMENTS

  • You need the a-Shell Mini app installed.
