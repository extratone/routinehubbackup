Yes, another shortcut that combine multiple shortcuts into on but…

How is it different?

  • Broad compatibility – Works on iOS 16 as well as macOS.
  • Straightforward process – Choose a main shortcut, then choose any number of shortcuts and where they get inserted.
  • No pre-setup required – It can insert shortcuts in the beginning or end of the main shortcut or right after any comment actions.
  • Shortcuts inserted in the beginning or end can optionally be put inside ‘If’ so that the whole block can move easily.
  • Same shortcut can be inserted multiple times without UUID conflicts.
  • Import questions in the main shortcut are retained.

How to use

  1. After choosing a main shortcut, you will be presented with a list of comment positions.
  2. Select a position and a shortcut. Repeat as many times as you want.
  3. Finish and you get a merged shortcut.
