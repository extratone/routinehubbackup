HiRes Twitter can download media from public or private accounts at the highest quality, save to Photos or Files or share. It can download multiple media types in a single tweet. It can also upload images at up to 8K.

It requires the free a-Shell mini app to download highest quality GIFs or set the media date to the date tweeted.

Downloading Media

  • Long-press a tweetâ€™s share button, then select HiRes Twitter from the list.
    • Alternatively you can tap the share button, tap â€˜Share viaâ€™, then select HiRes Twitter. This may not work on private tweets, though.
  • Images are downloaded at up to 8K (for comparison, the Twitter app is limited to 4K).
  • GIFs can be downloaded as GIF and/or mp4. Since Twitter stores submitted GIFs as mp4, this shortcut uses the same algorithm as Better GIF Makerâ€˜s to obtain maximum GIF quality (optional).
  • You can save media to iCloud or On My Device.

Uploading Images

  • To compose a tweet with images at up to 8K (for comparison, the Twitter app is limited to 4K), run HiRes Twitter from the Shortcuts app.
  • If you share your own tweet, you can reply to it with images.

User Settings

  • You can change user settings by running HiRes Twitter from the Shortcuts app.
  • Connecting accounts enables you to download private media and upload images.
  • To change folder or album, toggle the Save to Files/Photos option off and on.
