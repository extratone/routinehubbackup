iOS 14
Now supports (and requires) WidgetPack version 1.0.0 (18) and later.
Version 1.0 and 1.1 will not be compatible with future versions of WidgetPack. That’s because WidgetPack’s Shortcuts actions are changing for the sake of simplicity.

You shouldn’t need to update your widgets from Album Widget (1.1) or (1.0) -- they should continue to work as long as you don’t need to set them up on a new device, and as long as you don’t run the shortcuts that generate those widgets once those actions are no longer available.
