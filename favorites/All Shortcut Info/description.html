Description

Welcome to All Shortcut Info, a Shortcut to get every available data about a Shortcut using it‘s Link

How to Use:

  • Share the Link of the Shortcut, either by copying it to your clipboard or by using the shortcut directly in the share sheet. This shortcut supports both RoutineHub and iCloud links.

  • Run the Shortcut, it will give you all the details about the shortcut.
  • Name, Creation Date, Last modified Date, iOS Version, Size
  • Optional: (prettified) Plist of the Shortcut, deviceID of the Creator, userRecordName of the Creator

  • It also creates a Shortcuts Icon, which the Shortcut will have after importing it
  • As well as downloading the Webpage Icon (Colors & Icons remain original)

  • At the end you get the option to download the Shortcut, also directly into the Shortcuts App

  • This shortcut is also customizable through a simple Dictionary at the top of the Shortcut. So you can tell it what to show and what not to show.

Hope you enjoy the Shortcut :D
