Description

Using WidgetPack, this shortcut lets you make a widget similar in function to iOS 13’s phone widget, but with the new look of iOS 14 widgets.

As this shortcut is a beta, has some limitations. Those are the following:

  1. Medium widget size only
  2. You must choose 4 contacts

Additionally, because Apple has neutered widgets in iOS 14, this app will have to launch WidgetPack before it prompts you with a call button. So that kinda sucks, but its not that bad.
