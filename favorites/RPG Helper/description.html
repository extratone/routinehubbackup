Description

Website and documentation: https://rpghelpersc.tumblr.com/

This is a shortcut with an entire RPG battle system that you can customize to your liking. Currently, it’s in beta stages- so it’s not very fully featured, but I’ll continue updating it from time to time. Please report any bugs or give me feedback! I want to make this tool the best it can be!

This shortcut currently includes:

• HP, SP, Attack, and Defense

• Being able to miss attacks.

• Support for items.

• Support for multiple enemies.

• Support for random damage numbers.
