Description

A simple utility for creating and parsing Discord Markdown timestamps.

A Discord timestamp is written like this:

<t:1627228800>

Discord renders the timestamp like this, in your local time zone (this example uses EDT):

July 25, 2021 12:00 PM

Adding a style code after the Unix timestamp tells Discord how to format the timestamp. (For example: <t:12345678:d>) By default, timestamps are displayed with the short date/time style, though several other style options are also available.

For more information on how dynamic timestamps and other custom Markdown formats work on Discord, refer to Discord’s developer documentation: https://discord.com/developers/docs/reference#message-formatting
