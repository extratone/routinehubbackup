Use SF Emoji Menu Builder to build vCard menus from SF Symbols and emojis. It supports:

  • Creating multiple vCard menu items at once
  • Images from SF Symbols or emojis
  • Custom color or multicolor SF Symbols
  • SF Symbol scaling and various weights

This shortcut requires the Scriptable app (free) to process SF Symbols. You can use apps such as SF Symbols Preview (free) to browse and find symbol names. iOS 13 users can only use emojis because Scriptable does not support SF Symbols on iOS 13.

Input Format

The shortcut has one dictionary action and one text action at the top. You can edit either one as input; JSON in text action takes precedence if it begins with a {. Input contains:

  • items: List of dictionaries for menu items. Each dictionary contains:
    • symbol: SF Symbol name or emoji, e.g. doc.text.fill
    • color: 6-digit hex RGB color for SF Symbols, e.g. ffffff or 23ec8d
    • title: Title of menu item
    • subtitle: Subtitle; optional
  • background: 6-digit hex RGB color for background
  • scale: Scale factor in % for SF Symbols; emojis are not affected
    • Change the value if you use big symbols such as bold.italic.underline or small ones like circled symbols.
  • style: Weight of SF Symbols; one of ultralight, thin, light, regular, medium, semibold, bold, heavy, black

When run, it will display a menu to show how it looks. If you tap any menu item, the vCard text will be copied to clipboard so that you can paste elsewhere.

Multicolor SF Symbols

SF Symbols contains a few multicolor ones such as wifi or weather symbols. They may look strange if you apply a custom color. To create single-colored versions, please try other menu tools such as SF Symbol for vCard Menu instead.
