What3Words v 1.0 (Jun 11, 2022)
Created by u/nicolobedendo on Reddit
———
What3Words is a shortcuts for convert current location in “3 word address” and copy to clipboard using the what3words API who is a fast and simple interface that allows you to convert 3 word addresses to coordinates and vice versa.

Before to run be sure to get and API Key from:
https://accounts.what3words.com/create-api-key?referrer=/public-api
and copy to your clipboard.

NOTE:
Even if the iOS App already have the widget I create this one for just copy to the clipboard the "3 word address".
