Description

I was looking for an app to be able to highlight quote from the web for a thesis. Couldn't find anything without registration so I made this little shortcut that save text (including link) to a markdown format.

I use Draft to save the text, but you could edit it and put your favorite markdown text editor that support append text (or just keep RTF and put it in a compatible app)
