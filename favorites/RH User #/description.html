Description

RoutineHub User Number

This shortcut uses the URL of a RoutineHub User’s profile picture to determine what number they were when they joined RoutineHub. If they don’t have a profile picture then the number will be unretrievable.

For example, I found out through this shortcut that I was the 1829th author to join RoutineHub.
