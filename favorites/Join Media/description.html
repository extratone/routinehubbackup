Description

OS Compatibility
macOS No
iOS 15 or higher Yes
iOS 14 Yes

Select multiple video or audio files, and BOOM you have a combined file. Quality remains the same because they are not re-encoded. Name and date will be inherited from the first one.

When selecting files from Photos, the order of selection will be applied. When selecting files from Files, the order will be sorted by file names.

This shortcut requires the free a-Shell mini app: https://apps.apple.com/app/id1543537943. a-Shell also works if you are on iOS 15.
