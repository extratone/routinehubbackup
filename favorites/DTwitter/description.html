Description

DTwitter lets you download the media from a public tweet.

This shortcut can

  • Download photos (Up to 8K resolution)
  • Download videos
  • Download HQ GIFs (The first shortcut that let you do this!)
  • Download tweets with multiple and various media (Videos, photos, and/or GIFs)

How to use DTwitter

It uses Twitter's API
