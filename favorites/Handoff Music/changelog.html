0.2 1 year, 4 months ago
iOS 15
Changes in this version include an error report when songs aren’t added, a way to easily fix those errors yourself and report them to me, and a warning if the queue it has on file is older than five minutes, which should help to ensure that you don’t accidentally load an old queue.

Unfortunately Beta 3 has not resolved the bugs in Extract text from Image, so for now selecting text by hand is still necessary.
The bug in “If [Item from List] ends with __” is also still present, so explicit tracks that contain “ E” anywhere in the title still won’t transfer.
Lastly, [Embed-a-Update] still isn’t working either, so there’s still no proper updater built in. It’ll remind you to check for updates starting in two weeks.
