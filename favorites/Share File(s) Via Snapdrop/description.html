Description

Snapdrop is a sublime alternative to AirDrop. It uses peer to peer connection in the same local network/wifi to send images, documents or text. As it is web based, it runs on every device.

This shortcut is perfect for sharing images and files from any destination directly via Snapdrop.

Overview

There are three total shortcuts to send content via Snapdrop directly from the share menu:

  1. share images and files
  2. share texts and URLs of any kind
  3. share files and folders as ZIP archive

Upon Installation you can change the domain if you are using another snapdrop instance or host your own

How to use:

Receive an item using Snapdrop

  • Open https://snapdrop.net

Send an item using Snapdrop

  • Open the item, then tap the Share button, Share, the More options button, or another button that displays the app’s sharing options.
  • Tap share file(s) via Snapdrop in the row of share options
  • Wait for the browser tab to open and tap the device name of a Snapdrop user on the same network.

Tip:

To have the shortcut always on top of the share menu, go to the bottom of any share menu, click "edit actions" and add the Snapdrop shortcut to your favorites by tapping the green plus sign.

Tip #2:

Snapdrop is a webapp and can be installed on all machines as a cached version.


How it works

This shortcut implements its own Snapdrop front end by putting the source files directly into the shortcut, adding the items to be shared as json to a div data-attribute in the html page and base64encode the complete page in order to open it as a data-uri.

The code can be found at https://github.com/schlagmichdoch/snapdrop-via-ios-shortcuts

I found the technique as it is used to create this great iPad calculator shortcut: https://www.icloud.com/shortcuts/27ce14ff62724cca8e63cd6c5dbec30c
