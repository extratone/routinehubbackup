This is a two-action snippet used in SF Emoji Menu Builder for emoji to image conversion.

  • The first action draws an emoji at the center of a webpage, the second action crops it.
  • You can change the background color with an RGB hex value (#ddeeff in the example). It doesn't support transparent background.
  • 48 seems to be the minimum font size for maximum quality; anything bigger won't give you a better image.

Please use it freely in your shortcuts. :)
