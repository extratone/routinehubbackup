Description

This shortcut allows the user to visually select a fragment of a picture by overlaying a frame, and returns the size and position of the selected fragment in dictionary format.

  • This shortcut requires the a-Shell mini app
  • The shortcut doesn’t support the case of rotating the frame
  • The frame is embedded in Base64 in the shortcut and it’s easy to replace
  • The shortcut works as a function if no input is received, otherwise works as a demo, showing the cropped fragment with the info as its title
