1.1 1 year, 11 months ago
iOS 14
HighResArtwork. just received its first ever update adding lots of new features

This update includes:
**Functionality:**
- Ability to download the Screenshots you see under an AppStore app (needs to be turned on in settings).
- If a SoundCloud song is not found on Apple Music it will now download the SoundCloud artwork instead.
- You can now make the shortcut skip any menu automatically (enabled in settings)
- Shazam will let you retry if it can't find a song (iOS 14)
- If the Artist Artwork breaks (has happened temporarily a few times) there is now a backup method using the shazam website.
- Shortcut can now download artwork from a shazam webpage
- If you are on iOS 14, the Shortcut will no longer open the app if started elsewhere

**Visuals:**
- Subtitles are turned off if they don't show important information (can still be enabled through settings)
- Shazam has option to show the listening animation (turned off as the graphic seems to interfere with the following menu causing the app to crash).
