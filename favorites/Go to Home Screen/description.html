Description

iOS 15: Works (iOS 15.1 beta 2)

This shortcut combines actions from iOS 12-15 and macOS 12 to go to the Home Screen/LauchPad.

It can be used as a replacement of a single "Open Springboard" action in an iOS 12-14 shortcut, in order to avoid the "can't open app" error in iOS 15. I found about this issue with my Frame Screenshots shortcut.
