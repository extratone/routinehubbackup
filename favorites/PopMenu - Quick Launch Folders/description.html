Description

After the release of iOS 15, I really missed the ability to create quick launch folders in shortcuts. So, I have created this shortcut that works in iOS 15 (and hopefully 16 too).

Note: This shortcut requires PopMenu Manager, which will automatically be installed by PopMenu.

Features

  • Quick access to apps, shortcuts, images, files, links, contacts and notes
  • Edit and rearrange folders at any time
  • Automatic icon search for apps, shortcuts, websites and contacts
  • Replace default icons with any image at any time
  • Designed for both light and dark modes
  • Each new folder does not require its own shortcut
  • Opens in about a second on newer devices

Use cases

  • Launch from the dock, from widgets, with back taps or keyboard shortcuts
  • Quick access to documents like vaccine cards
  • Free up space on your Home Screen

Not so great

  • Managing folder layout is tedious but requires no shortcut editing
  • For performance reasons, two shortcuts are used in total
  • Light mode makes some icons more difficult to see despite shadow effects

Permissions

  • PopMenu needs permission to access your launch items.
  • RoutineHub updates require network access
  • PopMenu needs permission to run other shortcuts
  • iCloud Drive permissions are needed for folder storage
