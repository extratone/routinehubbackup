Glyph Search allows searching for shortcut glyphs (the symbols in the icons) by name in any of the 39 languages of the Shortcuts app.

Its result is an scroll-able glyph grid with the searched glyph highlighted by the intersection of a vertical and an horizontal color bands.

New version 2.0!

Version 2.0.0 updates the glyphs to the iOS 15.4 / macOS 12.3 ones, including folder glyphs, and shows the location of glyphs imitating each device's layout (horizontal for iOS, vertical for macOS; four rows of folder glyphs in iPhones, three rows in iPads).

iOS 15 loss of performance (relatively to iOS 14) made the original search (made with native actions) crash frequently, and alternative methods were too slow, so the search is now done with JavaScript.

JavaScript's speed also allows significant improvements in the order of the search results and the matched text shown at the end.

Previous versions

Version 1.1.0 improved the final visualization by showing both a zoomed and a general view of the glyphs. Both views are positioned so the scrollbar marks what part of the general view is zoomed above.

The glyph images are from a stitched screenshot of a 3x iOS device. All the glyph data is also available in my Shortcuts repo @ GitHub.
