Description

When you run across articles you’d like to link to in a blog post, this shortcut makes it quick and easy to get a template blog post ready with its information. It includes the author, publication, URL, (optional) blockquote, and (optional) your additional comments.

It saves the resulting Markdown to Drafts with tags for blog and linked.

The shortcut could be adapted to save the text to another notes app.

Latest Release Notes

2.1 - Feb. 3, 2022, 6:45 p.m.

- Added separators around metadata at the bottom of the draft.
- Adjusted formatting to include both an H1 header of just the linked article's title and a second line to be a Markdown link back to the article with a space to include the publication's name. This is better suited to how I format posts in either a link list or full-on linked article post. It makes it easy to delete the lines you don't need when editing later.
- Now you can read and edit the blockquote when adding comments. Before you couldn't only read, not edit, and longer blockquotes could get cut off.
- If nothing is passed into the shortcut, it will ask for text where you can paste in a URL
