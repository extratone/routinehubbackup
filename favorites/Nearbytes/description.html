Description


Nearbytes utilizes the power of the Google Places API and the iPhone’s Location feature to find anything you want wherever that may be! Users can also set parameters such as ‘Open Now’, minimum and maximum price, and search radius.

As of the release of v3, users can now view additional details about a particular place, such as full address, phone number (with added 'Tap to Call' feature), business hours, user ratings & reviews, and more!

To view place details:

  1. Tap on the name of the place in the search results table
  2. Tap the share icon at the bottom of the screen
  3. Tap 'Copy' in the share sheet menu
  4. Tap 'Shortcuts' in upper left-hand corner of your screen

This will take you back to Shortcuts where the additional details page will be displayed!

Note

Nearbytes requires a Google Places API key to operate. If you do not have one, you can obtain a FREE Google Places API key here.
