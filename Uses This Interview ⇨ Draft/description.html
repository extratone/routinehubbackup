Description

This shortcut utilizes the Uses This API to gather properly-formatted Markdown contents of individual Uses This interviews - along with an index of the software/hardware tools mentioned - to create a new draft in Drafts.

It was designed to do so from "natural language" (First Last) name input, but I plan to continue making other variations that accept interview URLs as input as well as some for other markdown-based notetaking/word processing/knowledge management applications.

Note: because setup steps appear not to be functioning in either of the current iOS/iPadOS 16.1 betas, you will need to manually specify personalized configuration (like tags to attach to each draft.)
