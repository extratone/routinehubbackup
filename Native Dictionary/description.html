Description

This shortcut utilizes the native Show Definition action and its integration with Apple's dictionary services to create rich-formatted Apple Notes for inputted terms (by way of the share sheet or text input) by passing the desired content through the system clipboard

At install, you'll be prompted to select an Apple Notes folder in which to store new definition notes as well as any Apple Notes tags you'd like to attach to each note. If you do not wish to attach any tags, you'll need to delete the action or be prompted at each run to select them.

Instructions

  1. Provide a single term to define (either by way of the share sheet or when prompted for text.)
  2. Select your preferred definition from the Show Definition action's popup.
  3. Copy the portion (up to ⌘A to Select All) of the result you'd like to save.
  4. The contents of the system clipboard will be added to a new Apple Note.
