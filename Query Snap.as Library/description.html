Description

This shortcut uses the Snap.as API to retrieve one’s entire Snap.as library in JSON format. By default, a Set Name action sets the name of the resulting file to snapas.json before prompting the user with a Save File action.

Upon installation, you will be required to set your Write.as/Snap.as user token.
