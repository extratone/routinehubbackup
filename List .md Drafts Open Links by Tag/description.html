Description

This shortcut creates a new draft with a (-) list of drafts from a single tag search, hyperlinked with their respective Drafts Open URLs. By default, the new draft is titled in this format: [tag] - [DavodTime™]. Upon installation, you will be asked to specify a maximum number of drafts to return in the search (optional,) as well as any tags you'd like to apply to the new draft (also optional.)

Upon running the shortcut, you'll need to specify a single tag to search in an Ask For Input action.
