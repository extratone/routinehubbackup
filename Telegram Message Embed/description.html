Description

This shortcut receives the URL for a single Telegram message (ex: https://t.me/extratone/10876) and constructs standard format HTML embed code, which it then outputs directly with a Stop and Output action so that (in theory,) one can use it to replace selected URLs immediately from the share sheet.
