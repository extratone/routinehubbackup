Description

This shortcut automates the process of uploading to The Null Pointer with curl using the first command listed on that landing page, verbatim:

curl -F’[email protected] Input (File Path)’ https://ttm.sh

Here’s an example result: https://ttm.sh/iad.43.png

It includes by default an action to save the result in a Data Jar specified at install, but this is not required. If you do not intend to use it, you will need to delete the action, though.

Here’s a very brief video demo.

For more information on the service, see its repository. If you’d like to know more about the Tildeverse, see Paul Ford’s “I had a couple drinks and woke up with 1,000 nerds” and/or come join me on Tilde.Town. :)
