Description

This shortcut uses the Write.as API to retrieve the markdown-formatted body text of a post at the inputted URL and copies it to the system clipboard. It works best when added to the share sheet.

Familiar Shortcuts users might take it upon themselves to add actions for their notetaking/thought management application(s) of choice at the end as I've done with Drafts.
