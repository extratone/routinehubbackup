Description

This shortcut uses the new actions present in Iconfactory’s Tot and Tot Pocket to transfer contents and metadata of a Dot specified at run to a new draft in Drafts in the format listed below.

Upon installation, you’ll be asked whether or not you’d like to add tags by default to the new draft.

Text

- Words: Dot Info (Word Count)
- Characters: Dot Info (Character Count)
- Modified: Dot Info (Modification Date)
